# NX

```bash
# create package
yarn nx <package_name>

# nx command
yarn nx [-bdirwhV] <package_name>
# -b   build
# -d   delete
# -i   install
# -r   remove build directory
# -w   watch current package
# -h   help
# -V   version

# ---------------------------

yarn clean # package/{lib,esm} & storybook-static

yarn storybook # dev
yarn storybook:build
```
